This is an introduction to CSS. We are learning how to link an external CSS file, use selectors to style elements, and manipulate basic layout properties like margin, padding, and borders.

CSS stands for Cascading Style Sheets. It is used to control the layout and presentation of HTML elements. By separating content (HTML) and design (CSS), we can maintain clean, organized, and scalable websites.
